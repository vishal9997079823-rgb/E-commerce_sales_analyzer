HOW TO SET UP AND RUN
──────────────────────

STEP 1 — Install Python
  Download Python 3.8+ from: https://www.python.org/downloads/

STEP 2 — Install Pandas
  Open terminal / command prompt and run:
    pip install pandas

STEP 3 — Run the Program
  Make sure sales_data.csv is in the same folder, then:
    python sales_analyzer.py

STEP 4 — View Output
  - Report prints on the screen
  - Also saved automatically to: sales_report.txt

============================================================
FEATURES
  1. Loads raw CSV sales data
  2. Cleans data (removes empty rows, fixes data types)
  3. Calculates total revenue, return rate, avg order value
  4. Shows best-selling products ranked by revenue
  5. Shows monthly revenue breakdown
  6. Compares Meesho vs Flipkart performance
  7. Shows return rate by product
  8. Gives business recommendations
  9. Saves full report to sales_report.txt automatically
